# Growth Experience Review (G.E.R) v0.1.0

## Highlights

- Export Codex archived session JSONL files into Markdown.
- Analyze sessions by week and generate mentoring-style reports.
- Build an HTML dashboard with:
  - latest summary
  - six-axis comparison graph
  - growth trend chart
  - past report browser
- Keep generated personal outputs outside version control by default.

## Intended audience

- Codex users who are not developers
- people building a weekly reflection habit
- users who want practical mentoring prompts from their own session history

## Notes

- This release does not include a public license yet.
- Generated reports are local-only by default.
